# 記錄點 #09: 勾選清單面板

**優先級**: P2
**狀態**: 待開發
**對應測試**: `tests/e2e/user-journey/09-check-list.spec.ts` (待建立)

---

## 概述

勾選清單面板，顯示已選擇的鴿子列表，提供單個刪除和全部刪除功能。

---

## 進入方式

- 前置記錄點: #1 → #2 (需先選擇鴿子)
- 操作步驟:
  1. 在賽事詳情選擇至少一隻鴿子
  2. 點擊「勾選清單 N」按鈕展開

---

## DOM 結構與選擇器

| 區域 | 元素 | 選擇器 | 功能 |
|------|------|--------|------|
| 按鈕 | 展開/收起 | `button "勾选清单 N"` | 切換面板 |
| 列表 | 鴿子行 | 排名 + 環號 + 舍號 | 顯示已選鴿子 |
| 列表 | 刪除單個 | `button "删除"` | 移除單隻鴿子 |
| 底部 | 全部刪除 | `button "全部删除"` | 清空清單 |

---

## 面板結構

```
┌─────────────────────────────────┐
│ 勾選清單 2                  [v] │
├─────────────────────────────────┤
│ 排名 │ 公環        │ 操作      │
│  1   │ 15-1004999  │ [刪除]    │
│      │ 舍号: 1578  │           │
│  2   │ 27-0162950  │ [刪除]    │
│      │ 舍号: 2127  │           │
├─────────────────────────────────┤
│         [全部刪除]              │
└─────────────────────────────────┘
```
